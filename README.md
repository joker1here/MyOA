# 网上办公系统  
由肇庆学院19软件1班制作  
## 1.背景  
项目名称：网络办公自动化系统（OA）  
Web设计与开发课程项目
## 2.系统定义  
网络办公自动化系统（OA）是针对小型企事业单位设计的一款通用办公系统，其主要用户包括管理员、部门领导和普通员工三类，主要完成各类信息的管理、个人计划的制定、文件的收发及公告的发布及查看。管理员可以完成个人信息、员工信息、部门信息和公告信息的管理，部门领导可完成个人信息、员工信息、活动信息和公共信息的管理，而普通员工只能完成个人信息的管理、活动信息和公共信息的管理。
